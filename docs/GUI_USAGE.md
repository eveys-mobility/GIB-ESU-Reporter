# GUI Kullanım Kılavuzu

1. Mali mührü bilgisayara takın.
2. Uygulamayı açın.
3. VKN, unvan ve EPDK lisans numarasını girin.
4. AKİS PKCS#11 yolu otomatik gelmezse `Otomatik Bul` veya `Seç` kullanın.
5. `Alias Tara` ile sertifika alias listesini alın.
6. Genellikle `SIGN0` alias'ını seçin.
7. Excel dosyasını seçin.
8. Dönemi seçin.
9. PIN girin.
10. `Hazırla ve Test Et` butonuna basın.
11. Test sonucu `success30` ise `Canlıya Gönder` aktif olur.
12. Canlı gönderim için ekranda istenen onay metnini yazın.
13. Canlı status `success30` ise işlem tamamdır.

PIN kaydedilmez. İşlem çıktı dosyaları arşiv klasöründe tutulur.
