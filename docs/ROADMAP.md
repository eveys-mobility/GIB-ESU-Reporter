# Yol Haritası

## Tamamlanan temel özellikler

- Excel okuma
- Kolon algılama
- Plaka normalizasyonu
- Plaka bazında toplama
- GİB EŞÜ XML üretimi
- Gömülü XSD doğrulama
- PKCS#11 mali mühür entegrasyonu
- XAdES-BES XML imzası
- Yerel XML imza doğrulama
- ZIP paketleme
- SOAP WS-Security gönderimi
- Test/canlı ortam desteği
- JavaFX masaüstü arayüz
- Otomatik SSL truststore hazırlama

## Planlanan geliştirmeler

- Daha okunabilir GİB durum kodu ayrıştırma
- Geçmiş gönderim ekranı
- Aynı dönem tekrar gönderim uyarısı
- Otomatik güncelleme mekanizması
- Detaylı audit raporu ekranı
- Kurumsal imzalı macOS/Windows installer
