# Mimari

Eveys GİB EŞÜ Reporter, aynı iş katmanını kullanan CLI ve JavaFX masaüstü arayüzünden oluşur.

```text
Excel -> XML -> XAdES-BES imza -> ZIP -> SOAP send/status
```

Paketler:

```text
dev.eveys.gibesu.audit      Audit dosyaları
dev.eveys.gibesu.cli        Terminal komutları
dev.eveys.gibesu.config     Ayar modeli
dev.eveys.gibesu.desktop    JavaFX arayüz
dev.eveys.gibesu.excel      Excel okuma
dev.eveys.gibesu.gib        SOAP client ve paketleme
dev.eveys.gibesu.model      Veri modelleri
dev.eveys.gibesu.sign       PKCS#11 / XAdES-BES imza
dev.eveys.gibesu.util       Yardımcı sınıflar
dev.eveys.gibesu.verify     XML imza doğrulama
dev.eveys.gibesu.xml        GİB XML üretimi ve XSD doğrulama
```

CLI teknik kullanım ve otomasyon için; masaüstü arayüz ise terminal kullanmayan personel için tasarlanmıştır.
